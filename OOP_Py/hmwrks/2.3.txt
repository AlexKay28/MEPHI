Ќапишите программу на €зыке Python, читающую со стандартного ввода натуральное число и печатающую
на стандартный вывод количество его натуральных делителей.
 омментарий: остаток r при делении целого числа a на целое число b в €зыке Python (как и во многих
других €зыках) можно найти с помощью оператора %: r = a % b.


#Third task
Number = int(input())
for i in range(1, Number + 1):
    if Number % i == 0:
        print(i)
